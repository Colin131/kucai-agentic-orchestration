# Decisions

> PM 写。记录 PRD / 技术选型 / scope / 招聘 决定。append-only。

## 文件命名

`YYYY-MM-DD_<topic>.md`

例：

- `2026-06-26_template_design.md`
- `2026-07-15_hire_data_agent.md`
- `2026-08-01_db_choice_postgres.md`

## 模板

见 [_TEMPLATE.md](_TEMPLATE.md)。

## 不应该出现在这里

- 复盘 → `journal/post_mortems/`
- review 报告 → `journal/reviews/`
- QA 报告 → `journal/qa-reports/`
- 派工合同 → `goals/`
- 累积教训 → `memory/eng-lessons.md`
